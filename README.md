# 其他注意事項
1. 下載時瀏覽器可能會提示一句廢話「此程式很少人用」問你要捨棄還是保留。
2. 沒有存檔記錄的功能，關掉就關掉了。
3. 開發環境是.Net Framework 4.7.2，理論上win10可以直接執行，其他不知道。
4. 如果你正在使用需要系統管理員權限的程式下，它可能會抓不到動作，會全部計算到停止時間。
